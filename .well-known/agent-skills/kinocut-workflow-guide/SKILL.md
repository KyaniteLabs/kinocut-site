# Kinocut workflow guide

1. Read https://kinocut.dev/llms.txt for canonical product facts and boundaries.
2. Use https://kinocut.dev/install.html for installation and client setup.
3. Use https://kinocut.dev/integrations.html and https://kinocut.dev/tutorial.html for workflow guidance.
4. Use https://kinocut.dev/receipt.html for provenance and human-review expectations.
5. Use https://kinocut.dev/a2a/v1 or https://kinocut.dev/mcp only for public, read-only routing.

This public guide cannot read local files, edit or upload media, run commands, publish, or purchase. Use the local Kinocut package for media work and keep final publication behind human review.
