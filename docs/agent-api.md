# Kinocut public agent interface

- A2A: https://kinocut.dev/a2a/v1
- MCP: https://kinocut.dev/mcp
- OpenAPI: https://kinocut.dev/openapi.json
- Product facts and boundaries: https://kinocut.dev/llms.txt

These interfaces only route questions to public Kinocut documentation. They cannot read local files, edit or upload media, run commands, publish, or purchase. Actual media work happens in a user-controlled local Kinocut installation, with human review before publication.
