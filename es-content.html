Kinocut en español

Kinocut (antes mcp-video) es un servidor MCP, biblioteca Python y CLI kino para edición de video local con barreras, Video Receipts y controles de calidad. Gratis, Apache-2.0.

Instalación

brew install ffmpeg   # o: sudo apt install ffmpeg
pip install kinocut
kino doctor
claude mcp add kinocut -- uvx --from kinocut kino

Ruta dorada (clone): python scripts/golden_path.py — ver GOLDEN_PATH.md.

Preguntas frecuentes

  • ¿Es gratis y local? Sí. Núcleo Apache-2.0; el material no se sube a una nube de Kinocut.
  • ¿Qué agentes? Claude Code, Cursor y clientes MCP con stdio local.
  • ¿Cuántas herramientas? 135 en la 1.7.0 publicada; la punta de desarrollo puede ser mayor (no es 1.8 publicado).
  • ¿Antes mcp-video? Sí; compatibilidad durante la ventana documentada.
